# Changelog

## [1.1.0] - 2025-02-10

### Fixed
- SVG mime type detection for WordPress 4.7.1+
- Improved SVGZ compression handling

### Added
- Enhanced XXE protection for PHP 8.0+
- Better error messages for upload failures

### Improved
- Code organization and documentation
- Settings page user experience

## [1.0.0] - Initial Release

### Added
- SVG and SVGZ file upload support
- Metadata stripping from design applications (Adobe, Figma, Sketch)
- ViewBox auto-correction for responsive scaling
- Media Library thumbnail rendering fixes
- Admin-only upload restriction (configurable)
- Minimal frontend footprint
